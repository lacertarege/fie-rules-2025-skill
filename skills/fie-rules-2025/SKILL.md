# FIE Rules 2025

Use this skill to answer rule questions under the official FIE ruleset in force on the FIE rules page as of December 2025.

## Quick Start

- Identify whether the user is asking about bout procedure, equipment, competition structure, or penalties.
- Open [references/official-documents.md](references/official-documents.md) first to pick the correct official source.
- Use [references/topic-routing.md](references/topic-routing.md) to map common question types to the right document.
- Cite the document name and, when available, the article number mentioned in the source being used.

## Source Selection

Use the official FIE documents in this order:

1. Use the Technical Rules for weapon actions, priority, valid target, piste rules, bout procedure, referee commands, timing, and general conduct on piste.
2. Use the Material Rules for weapons, uniforms, mask, bodywire, safety requirements, and equipment conformity.
3. Use the Organisation Rules for pools, direct elimination, team format, event structure, and competition administration.
4. Use `t.170` together with the linked article when the question is specifically about cards, offences, or penalties.

## Answering Rules Questions

- Prefer the official FIE wording over memory when the user asks for an exact rule.
- Keep weapon distinctions explicit. Do not generalize a foil rule to epee or sabre.
- Distinguish between a direct citation from the rulebook and an inference or plain-language explanation.
- If the user asks for the "latest" rule rather than the 2025 rule set, verify whether the FIE page has changed before answering.
- If the question could depend on a national federation adaptation, say clearly that this skill covers FIE international rules, not local competition rules unless the user asks for another federation.

## Common Request Patterns

- "What is the rule for right of way in sabre?" -> Technical Rules
- "Is this glove or mask legal?" -> Material Rules
- "How do pools and DE work?" -> Organisation Rules
- "What card applies for leaving the piste?" -> `t.170` plus the cited technical article
- "Summarize the FIE rules for beginners" -> Start from Technical Rules, then add Material and Organisation only as needed

## Output Style

- For quick questions, answer in plain language first and then cite the relevant FIE source.
- For study help, provide a short summary grouped by weapon or topic.
- For disputes or edge cases, point to the official document link and name the article if found.
