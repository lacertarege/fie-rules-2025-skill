# FIE Rules 2025 Skill

Reusable Codex-compatible skill for answering questions about the official FIE fencing rules in force on the FIE rules page as of December 2025.

## Included Skill

- `fie-rules-2025`: covers technical rules, material rules, organisation rules, and `t.170` penalties

## Install

```bash
npx skills add https://github.com/lacertarege/fie-rules-2025-skill --skill fie-rules-2025
```

## Repository Layout

```text
skills/
  fie-rules-2025/
    SKILL.md
    agents/openai.yaml
    references/
```

## Sources

The skill points to the official FIE rules page and the December 2025 rulebooks published by the FIE.
